Was wriyyen first computer virus - Creeper, was an experimental computer program written by Bob Thomas at BBN in 1971. Its original iteration was designed to move between DEC PDP-10 mainframe computers running the TENEX operating system using the ARPANET, with a later version by Ray Tomlinson designed to copy itself between computers rather than simply move.

Archie is considered the first Internet search engine. The original implementation was written in 1990 by Alan Emtage, then a postgraduate student at McGill University in Montreal, Canada.

1991

The World Wide Web was invented by English computer scientist Tim Berners-Lee at CERN, and originally conceived as a document management system. The first proposal was written in 1989, and a working system implemented by the end of 1990 including the World Wide Web browser and an HTTP server. The technology was released outside CERN to other research institutions starting in January 1991, and then to the general public on 23 August 1991.
